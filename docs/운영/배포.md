## 트러블슈팅

(stub)
| 증상 | 원인 | 해결 |
|------|------|------|
| 시작 실패 (OOM) | heap 부족 | `MaxRAMPercentage` 조정 또는 plan 업 |
| `/auth/*` 404 | file mode | `SIGNAL_DESK_STORE_MODE=jdbc` |
| DB connect fail | Postgres 어태치 안 됨 | Railway 어태치 확인 |
| CORS 403 | env 누락 | `SIGNAL_DESK_CORS_ALLOWED_ORIGIN_PATTERNS` 추가 |
| JWT secret 더미 경고 | env 미설정 | `openssl rand -base64 48` 으로 새 값 |
| 외부 API timeout 폭증 | 외부 장애 | fallback 동작. 로그 확인 |
