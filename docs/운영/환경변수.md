# 환경변수 (백엔드)

> **누가 봄**: 운영자
> **언제 봄**: 새 환경 셋업 / 변수 추가

## Spring Boot @Value 매핑

Spring 은 `signal-desk.foo.bar` ↔ `SIGNAL_DESK_FOO_BAR` 자동 매핑.
실제 application.yml 의 키는 dot 표기, 환경변수는 SCREAMING_SNAKE.

## 필수

(stub)
| 키 (yml) | 환경변수 | 의미 | 운영 예시 |
|---------|---------|------|----------|
| `signal-desk.store.mode` | `SIGNAL_DESK_STORE_MODE` | 저장 모드 | `jdbc` |

## 외부 API 키 / URL (옵션)

비어있으면 해당 기능만 비활성(noop). URL 은 모두 default 가 있어 보통 키만 넣으면 된다.
| 환경변수 | 의미 | default |
|---------|------|---------|
| `FRED_API_KEY` | FRED Web API 키 (CPI/금리 등 매크로). 없으면 매크로 카드 비활성 | (없음) |
| `FRED_API_BASE_URL` | FRED Web API base | `https://api.stlouisfed.org/fred` |
| `DART_API_KEY` | OpenDART 키. 없으면 KR 공시 scan noop | (없음) |
| `DART_BASE_URL` | OpenDART base | `https://opendart.fss.or.kr` |
| `GEMINI_API_KEY` | Google AI Studio 키. 없으면 미디어 요약/브리프 비활성 | (없음) |
| `GEMINI_BASE_URL` | Gemini base | `https://generativelanguage.googleapis.com` |
| `GEMINI_MODEL` | Gemini 모델 | `gemini-2.0-flash-exp` |
| `FINNHUB_API_KEY` | finnhub.io 키. 없으면 실적 캘린더 비활성 | (없음) |
| `FINNHUB_BASE_URL` | finnhub base | `https://finnhub.io/api/v1` |
| `SEC_EDGAR_TICKERS_URL` | SEC EDGAR 티커 매핑 URL | `https://www.sec.gov/files/company_tickers.json` |
| `SEC_EDGAR_RECENT_URL` | SEC EDGAR 최근 공시 URL | `https://www.sec.gov/cgi-bin/browse-edgar` |
| `SEC_EDGAR_USER_AGENT` | SEC fair-access 용 user-agent (연락처 포함 필수) | `signal-desk-personal gwim113000@gmail.com` |
| `YAHOO_FINANCE_BASE_URL` | Yahoo Finance screener base | `https://query1.finance.yahoo.com` |
| `NAVER_GLOBAL_BASE_URL` | Naver Global (US 시세) base | `https://api.stock.naver.com` |

## 외부 API 활성화 (옵션)

`application.yml` default 가 모두 enabled (`venue-signals` 만 false). 비활성하려면 false 설정.
이 `enabled` 키들은 yml 에 `${ENV}` 플레이스홀더가 없어 Spring relaxed-binding 으로만 오버라이드되며, **환경변수에서는 하이픈이 제거**된다 (`sec-edgar` → `SECEDGAR`).
| 키 | 환경변수 |
|----|---------|
| `signal-desk.integrations.krx.enabled` | `SIGNAL_DESK_INTEGRATIONS_KRX_ENABLED` |
| `signal-desk.integrations.naver.enabled` | `SIGNAL_DESK_INTEGRATIONS_NAVER_ENABLED` |
| `signal-desk.integrations.cboe.enabled` | `SIGNAL_DESK_INTEGRATIONS_CBOE_ENABLED` |
| `signal-desk.integrations.fred.enabled` | `SIGNAL_DESK_INTEGRATIONS_FRED_ENABLED` |
| `signal-desk.integrations.google-news.enabled` | `SIGNAL_DESK_INTEGRATIONS_GOOGLENEWS_ENABLED` |
| `signal-desk.integrations.venue-signals.enabled` | `SIGNAL_DESK_INTEGRATIONS_VENUESIGNALS_ENABLED` (기본 false) |
| `signal-desk.integrations.dart.enabled` | `SIGNAL_DESK_INTEGRATIONS_DART_ENABLED` |
| `signal-desk.integrations.gemini.enabled` | (gemini 는 enabled 토글 없음 — 키 유무로 활성) |
| `signal-desk.integrations.finnhub.enabled` | `SIGNAL_DESK_INTEGRATIONS_FINNHUB_ENABLED` |
| `signal-desk.integrations.sec-edgar.enabled` | `SIGNAL_DESK_INTEGRATIONS_SECEDGAR_ENABLED` |
| `signal-desk.integrations.yahoo-screener.enabled` | `SIGNAL_DESK_INTEGRATIONS_YAHOOSCREENER_ENABLED` |
| `signal-desk.integrations.naver-global.enabled` | `SIGNAL_DESK_INTEGRATIONS_NAVERGLOBAL_ENABLED` |

## 부팅 시 검증

(stub)
- JWT secret 더미값 → `[SECURITY] ERROR` 로그
- JWT secret 32자 미만 → `[SECURITY] WARN` 로그
- Google audiences 미설정 → `[SECURITY] WARN` 로그 (검증 스킵)

## secret rotation

(stub)
- JWT: 새 값 → env 갱신 → 자동 재시작 → 모든 사용자 재로그인 필요
- Google audience: 새 ID 추가 → env 콤마로 추가 → 재시작
