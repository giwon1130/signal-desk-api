## DB 스키마 (jdbc 모드)

마이그레이션 `src/main/resources/db/migration/V*.sql`. 주요 테이블·컬럼:

### 워크스페이스
- `signal_desk_watchlist` — 관심종목. (V9) `alert_below int`, `alert_above int`, `volume_alert boolean default false` 추가.
- `signal_desk_portfolio_positions` — 보유종목. (V8) `target_price int`, `stop_loss_price int` 추가 (선택 입력, NULL 허용).

### 알림 설정 (`signal_desk_alert_preferences`) — `user_id` PK
| 컬럼 | 기본 | 마이그 | 의미 |
|------|------|--------|------|
| `kr_enabled` | true | V12 | KR 알림 |
| `us_enabled` | false | V12 | US 알림 |
| `premarket_enabled` | true | V12 | 모닝 브리프 (08:30) |
| `composite_risk_enabled` | true | V15 | 합성 위험도 알림 (score≥8) |
| `market_preference` varchar(8) | 'BOTH' | V17 | UI 노출 시장 (KR/US/BOTH) |
| `evening_brief_enabled` | false | V18 | US 이브닝 브리프 (06:30 KST) |
| `midday_brief_enabled` | false | V22 | KR 장중 브리프 (12:30 KST) |
| `close_brief_enabled` | true | V22 / V23 | KR 마감 브리프 (15:40 KST). V22 false → V23 기본 ON 전환 |

### 공시 dedup
- `signal_desk_disclosure_seen` (V14) — OpenDART `rcept_no` PK, KR 공시 중복 방지.
- `signal_desk_us_disclosure_seen` (V16) — SEC EDGAR `accession_no` PK, US 공시 중복 방지.

### 모의투자 리그 (V19)
- `signal_desk_mock_league` — 시즌. `join_code` unique, `market_scope`(KR/US/BOTH), `currency`(KRW/USD), `starting_capital`, `started_at`/`ends_at`, `status`(DRAFT/OPEN/RUNNING/FINISHED), `visibility`(OPEN/CLOSED), `fee`, `max_position_pct`.
- `signal_desk_mock_participant` — `(league_id, user_id)` PK. `nickname`, `avatar_emoji`, `cash_balance`, `final_return_rate`, `final_rank`.
- `signal_desk_mock_trade` — **immutable 체결 원장** (UPDATE/DELETE 금지, 정정은 새 trade). `side`(BUY/SELL), `quantity`, `original_price`/`original_currency`, league 통화 환산 `price`, `exchange_rate`, `price_locked_at`, `fee_amount`, `notional_amount`. **Position 은 trade 합산으로 derived — DB 미저장.**
- `signal_desk_mock_reaction` — 거래 카드 이모지. `(trade_id, user_id, emoji)` PK, emoji ∈ 👍🔥😱🤔.

### 리딩 / 애널리스트 콜 (V21)
- `signal_desk_reading_leader` — `user_id` PK. `display_name`, `bio`, `invite_code` unique, `status`(PENDING/APPROVED/SUSPENDED).
- `signal_desk_reading_follow` — `(leader_user_id, follower_user_id)` PK 구독 관계.
- `signal_desk_reading_post` — 리딩 글. `title`, `body`, `visibility`(FOLLOWERS/PUBLIC).
- `signal_desk_reading_call` — 종목 콜. **`entry_price` immutable** (작성 시점 시세 박제, UPDATE 금지), `entry_currency`, `entry_locked_at`, `target_return_pct`(null=기본 +15%), `status`(ACTIVE/HIT/CLOSED), `hit_at`.

### 푸시 알림 로그
- `signal_desk_push_alert_log` — (V20) `direction` 컬럼을 varchar(8)→varchar(20) 확장 (PRICE_ABOVE/PRICE_BELOW/VOLUME_SPIKE 길이 수용).

## 정수 vs Double

(stub)
- 가격은 모두 `Int` (KR 원, US 센트 — 1원 단위 반올림)
- 등락률은 `Double` (percent — 4.6 = 4.6%)
- 신뢰도는 `Int` (0~100)
- 수량은 `Int`

US 가격 1원 단위 반올림 한계 — 현재 도메인이 KR 위주라 MVP 수용. 추후 분리 시 BigDecimal 검토.
